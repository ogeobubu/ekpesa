## üèõÔ∏è About Ekpesa Kingdom

Ekpesa Kingdom is a traditional Nigerian monarchy with an extraordinary heritage dating back to approximately **1200 AD** (13th century). Founded by Ekpesa, one of four brothers who accompanied Oranmiyan on his journey from the ancient Igodomigodo Kingdom (now Benin Kingdom), Ekpesa represents a living testament to Nigerian cultural continuity and traditional governance.

This official website serves as a comprehensive digital archive and cultural bridge, showcasing the rich history, vibrant traditions, and continued cultural significance of the Ekpesa people within the larger Okpameri communities across Akoko-Edo Local Government Area, Edo State, Nigeria.

## üöÄ Quick Start

### Prerequisites
- Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Web server (local development) or hosting platform
- HTTPS recommended for PWA functionality

### Local Development Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/ekpesa-kingdom-website.git
   cd ekpesa-kingdom-website
   ```

2. **Start a local web server**
   
   **Option A: Python**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   ```
   
   **Option B: Node.js**
   ```bash
   npx http-server -p 8000 -o
   ```
   
   **Option C: PHP**
   ```bash
   php -S localhost:8000
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```

### Performance Testing
```bash
# Run Lighthouse performance audit
npx lighthouse http://localhost:8000 --view

# Check Core Web Vitals in Chrome DevTools
# Performance tab -> Run Lighthouse
```

### Production Deployment

1. **Upload all files** to your web hosting provider
2. **Configure domain** (e.g., ekpesakingdom.netlify.app)
3. **Enable HTTPS** (required for PWA functionality)
4. **Configure GZIP compression** on your server
5. **Submit sitemap** to Google Search Console
6. **Set up monitoring** (Google PageSpeed Insights)

## üì± Progressive Web App (PWA)

This website functions as a Progressive Web App, providing:

- **Offline Functionality** - Core content accessible without internet
- **Add to Home Screen** - Install like a native app on mobile devices
- **App-like Experience** - Full-screen mode without browser chrome
- **Fast Loading** - Service worker for cached content delivery
- **Background Updates** - Automatic content updates when available
- **Push Notifications** - Ready for future notification features

### Service Worker Features
- **Intelligent Caching** - Cache-first for static assets, network-first for navigation
- **Background Sync** - Updates cached content when connection restored
- **Version Management** - Automatic cache cleanup and versioning
- **Update Notifications** - User-friendly notifications for new versions
